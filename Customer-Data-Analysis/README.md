## Customer Data Analysis

### Objective
To analyze customer purchasing behavior across gender, age groups, product categories, and payment methods to support data-driven business decisions.

### Tools Used
- SQL
- Power BI

### Key Analysis
- Gender-wise sales and revenue distribution
- Age group segmentation using CASE statements
- Category-wise purchase trends
- Payment method analysis
